# Career Compass Demo Script

## Demo Flow

### 1. Setup (Before Demo)
- Ensure all services are running:
  - Frontend: http://localhost:5173
  - Backend: http://localhost:5000
  - AI Service: http://localhost:8000
  - MongoDB: localhost:27017

### 2. Landing Page
- Open Career Compass homepage
- Explain the purpose: AI-powered resume analysis
- Show clean, intuitive interface

### 3. Upload Resume
- Click "Upload Resume" button
- Select sample resume (docs/sample-data/resume-sample.pdf)
- Show file preview/confirmation

### 4. Paste Job Description
- Copy sample JD from docs/sample-data/job-sample.txt
- Paste into text area
- Click "Analyze"

### 5. View Results
- **Score Card**: Overall match percentage
- **Gap Analysis**: Missing skills and qualifications
- **Recommendations**: Actionable tips to improve match
- **Skill Breakdown**: Matched vs missing skills

### 6. Interactive Chat
- Ask follow-up questions:
  - "How can I improve my Python skills?"
  - "What certifications would help?"
  - "How important is the 5-year experience requirement?"

### 7. History View
- Navigate to History page
- Show previous analyses
- Click to view past results

### 8. Settings (Optional)
- Configure API keys
- Adjust analysis parameters

## Key Talking Points

- **Speed**: Real-time analysis powered by LLMs
- **Accuracy**: Context-aware gap identification
- **Actionable**: Specific, personalized recommendations
- **Privacy**: Data stored securely, not shared
